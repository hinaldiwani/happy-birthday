## Audio Setup
To add the birthday song:
1. Place your birthday song audio file (MP3 format recommended) in the same folder as `index.html`
2. Rename it to `happy-birthday.mp3`
3. Or update the audio source in `index.html` to match your audio file name

**Free Birthday Music Sources:**
- YouTube Audio Library (royalty-free)
- Free Music Archive
- Incompetech
- Public domain music sites

## Files
- `index.html` - Main HTML structure
- `styles.css` - All styling and animations
- `happy-birthday.mp3` - Birthday song audio (not included - add your own)

## How It Works
1. Page loads with the cake and unlit candle on a light pastel background
2. After 5 seconds:
   - Candle lights with smooth flame animation
   - Background darkens gradually
   - Ambient glow appears around the cake
   - Birthday song starts playing
3. All effects are smooth and cinematic!
